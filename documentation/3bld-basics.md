## general

a letterpair es defining a corner 3 cycle. ther is only two letters givenn because the sticker C is tha same for all of them. E.g. LV means that it is solvable like this: 
C -> L
L -> V
V -> C
(so the scramble has to do the opposite)

---
## directions
U - up
D - down
F - front
B - back
L - left 
R - Right

---
## directions - letter
a corner can be diefined using 3 directkons e.g. URF is the corner having the tree faces up,right and front in common. 
This corner has a main Surface which is the first letter. so U is here the main suface. 
Using letther we would refere to it with the letter being on this sticker. 
letters are fixed on a sertain pieces and can bbe anywhere if the cube is scrambled. 
following: defines the position of the letther wen solfed, refffering to the positions mentioned:
